# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react/README.md) uses [Babel](https://babeljs.io/) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc) uses [SWC](https://swc.rs/) for Fast Refresh
 setup instructions
 1.run the command  "npm create vite@latest"  for setup the project enviroment
 2. select react framework and javacript
 3. run command "npm install" to install the dependency
 4. run command "npm axios" to install axios dependency for login compoments
  5. project run command is "npm run dev"
  6.  install mui fromework for button css using command "npm install @mui/material @emotion/react @emotion/styled"
  7. for routing we install "react-router-dom" using command "npm install react-router-dom"
